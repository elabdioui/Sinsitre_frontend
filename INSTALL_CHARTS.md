# Installation de Chart.js

Pour installer Chart.js et ses dépendances Angular, exécutez la commande suivante dans le terminal :

```powershell
npm install chart.js ng2-charts
```

Cette commande installe :
- **chart.js** : Bibliothèque de graphiques JavaScript
- **ng2-charts** : Wrapper Angular pour Chart.js

Après l'installation, redémarrez le serveur de développement Angular pour appliquer les changements.
